# 航空管制专业知识考试网页

## 项目简介
这是一个包含85道航空管制专业知识考试题目的静态网页，部署在GitHub Pages上。

## 内容特点
- 85道单选题，每题1分
- 基于航空管制标准和最佳实践的参考答案
- 响应式设计，适配手机和电脑
- 现代化UI界面

## 访问地址
https://bangikoko.github.io/aviation-exam-bangikoko/

## 技术栈
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6)

## 部署信息
- 部署平台：GitHub Pages
- 部署方式：通过GitHub Actions自动部署
- 访问方式：公开访问
